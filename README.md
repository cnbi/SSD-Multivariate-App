# SSD-Multivariate-CRT-App

This Shiny app for sample size determination for testing multiple outcomes in cluster randomised trials: intersection-union test, omnibus test, and homogeneity of treatment effects test. The results displayed come from the simulation study Methodology for Sample Size Determination for Testing Multiple Outcomes in Cluster randomized Trials by Barragan and Moerbeek (2026).

# Project

The results displayed are related to the project **Sample Size Determination for Cluster Randomised Trials with Multiple Outcomes: A Bayesian Hypothesis Testing Approach**. Further information about the study and methodology can be found in OSF <https://osf.io/g5ydk>

# Usage

There are two ways to access to the Shiny app:

1.  Online access via [SSD-Multivariate-App](https://cnbi.shinyapps.io/SSD-Multivariate-App/)
2.  Clone the repository and run the app locally from the `app.R` script.
